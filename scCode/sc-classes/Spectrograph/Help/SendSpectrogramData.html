SendSpectrogramData


Inherits from: DrawSpectrogram : Object

Purpose


Send fft magnitude array polled by PollFFT to openFrameworks for display of fft data. 


Scalling of fft data should be done in openFrameworks???


Usage


Add an instance of SendPectrogramData as dependant to PollFFT. 

PollFFT starts working as soon as the server boots and stops when the server stops. 

Example: 


//:Testing only: 

PollFFT('of') addDependant: SendSpectrogramData.new;


\test.playFunc({ LFSaw.ar(LFNoise2.kr(0.5).range(100, 1000), 0, 0.1) });


//:Starts automatically when the server starts

// Server.default.boot;


//:stops only when the server quites

Server.default.quit;
